# panel.222prod.cc

Это система для централизованного управления пользовательскими скриптами и удаленными серверами. Панель позволяет пользователям создавать и настраивать скрипты, отслеживать их состояние, управлять доступом и получать информацию о работе системы в реальном времени.
Административная часть включает управление пользователями, серверами, ключами, задачами планировщика, новостями и системными параметрами. На серверной стороне реализована интеграция с SSH/PM2 для удаленного выполнения команд, деплоя и контроля процессов, а также фоновые очереди и автоматические задачи (например, проверки, обновления и обслуживание). Итоговая цель проекта - дать единый интерфейс для безопасного и удобного жизненного цикла скриптов: от создания и настройки до запуска, мониторинга и поддержки.

## Что внутри

- `client` — фронтенд (React, TypeScript, Vite, Tailwind)
- `server` — backend API (NestJS, TypeScript, Prisma)
- авторизация, роли пользователей, админ-раздел, работа со скриптами и серверами

## Скрипты

### Client

- `npm run dev` — dev-сервер
- `npm run build` — production build
- `npm run test` — тесты

### Server

- `npm run start:dev` — dev-сервер NestJS
- `npm run build` — сборка
- `npm run test` — тесты
